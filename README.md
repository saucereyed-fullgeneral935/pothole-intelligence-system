## 🚗 What this app does

This app helps you inspect road images and find potholes with AI. It uses YOLOv8 to scan each image, mark potholes, and help you review road damage in a simple way.

It is built for real-world road photos, so it can handle different light, weather, and surface conditions. You can use it to look at single images or a batch of images on a Windows PC.

## 🗂️ What the app looks for

The system scans road photos for signs of damaged pavement. It checks for pothole shapes, rough edges, and areas that stand out from the road surface. It also aims to stay useful when the image has shadows, dust, glare, or poor road marks.

## ⚙️ Suggested setup steps

1. Keep the app in a folder you can find later.
2. Use clear road images for best results.
3. Close other heavy apps if your PC feels slow.
4. Use recent images from a phone, dashcam, or camera.
5. Make sure the image is not too dark or blurry.

## 📁 Input file tips

For best results, use images that:

- Show the road surface clearly
- Have the pothole in the frame
- Are not heavily cropped
- Use JPG or PNG format
- Have enough light to show road detail

If you use many images, place them in one folder before you scan.

## 🔧 If the app does not open

1. Check that the file finished downloading.
2. Extract the ZIP file if one was downloaded.

## 🧰 If image results look wrong

1. Use a clearer image.
2. Move the pothole closer to the center of the photo.
3. Avoid very dark or blurry pictures.
4. Try a different angle.
5. Use a higher-resolution image if possible.
